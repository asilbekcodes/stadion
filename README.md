# Sefera Stadium 

A Type-script , React Js and Vite based **football stadiums** website that provides real-time controlling __clients' orders and informations__ for football stadium holders.

## Table of Contents
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)



## Features
- Real-time client's updates
- 24/7 time working platform


## Installation

1. Clone the repo:
```bash
    git clone https://github.com/asilbekcodes/stadion.git
```


## Usage

1. Navigate to the project directory:
```bash
   cd stadion
```

2. npm package download: (<img src="https://cdn-icons-png.flaticon.com/512/5968/5968322.png" width="20">NodeJs is a must)
```bash
  npm i
```

3. Start the development server:
```bash
  npm run dev
```
